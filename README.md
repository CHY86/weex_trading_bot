# WEEX AI 智能交易機器人 (WEEX AI Trading Bot)

[English Version Below](#weex-ai-trading-bot-english)

這是一個針對 **WEEX 交易所**合約交易設計的自動化交易機器人。它結合了傳統技術分析指標（RSI、布林通道）與 **OpenAI (GPT-4o-mini)** 的人工智慧推理能力，旨在提供更具深度的交易決策。

此外，本專案包含一個強大的 **CLI 帳戶管理工具**，讓您可以快速查詢資產、管理倉位、調整槓桿以及執行緊急風控操作。

## ✨ 功能特點

### 🤖 智能交易核心 (`main.py`)
* **混合策略引擎**：結合技術指標（RSI 超買/超賣、布林通道突破）作為初步篩選，並將歷史 K 線數據提交給 AI 進行二次確認。
* **AI 深度分析**：使用 OpenAI `gpt-4o-mini` 模型分析最近 30 根 K 線的價格行為 (Price Action) 與型態，提供自然語言的交易理由。
* **風險控制**：內建交易冷卻時間 (Cooldown)、最大持倉限制與最大掛單限制，防止過度交易。
* **自動化日誌**：將 AI 的決策過程與交易執行結果上傳至雲端或本地日誌，方便回溯分析。

### 🛠️ 帳戶管理工具 (`check_account.py`)
一個全功能的命令行工具，用於監控與管理您的 WEEX 帳戶：
* **資產與持倉監控**：即時查看錢包餘額、未結盈虧 (PnL) 與詳細持倉數據。
* **訂單管理**：查詢當前掛單與歷史訂單記錄。
* **帳戶設定**：查看詳細帳戶資訊並**動態調整合約槓桿**。
* **緊急風控**：
    * **一鍵全平倉 (Close All Positions)**：市價平掉單一幣種或全帳戶持倉。
    * **撤銷所有掛單 (Cancel All Orders)**：一鍵取消所有未成交的委託單。
